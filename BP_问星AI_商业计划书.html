问题与机会Problem & Opportunity

😓 用户四大痛点
  • 门槛高：传统紫微斗数排盘复杂，专业命理师线下咨询动辄数百至数千元
  • 缺个性：通用 AI 无法基于真实命盘数据给出精准个性化分析，答案千篇一律
  • 体验差：现有命理 App 界面陈旧、交互枯燥，无法与年轻用户产生情感共鸣
  • 工具碎片化：排盘、六爻、合盘、运势查询分散在多个平台，体验割裂
🌟 市场机遇
  • 情绪经济崛起：Z 世代玄学消费持续升温，成为年轻人消费新热点
  • AIGC 赋能：AI 首次让大规模、个性化、低成本命理解读成为可能
  • 市场体量：国内命理市场超 500 亿元/年，数字化渗透率仍处于早期
  • 海外蓝海：全球 6000 万华人，AI 命理工具几乎空白，付费力是大陆 2-3 倍

产品与服务Product & Service

问星AI 是一款运行于浏览器端的全功能命理工具，无需下载，跨平台支持桌面与移动端。 依托精准的紫微斗数算法引擎与深度 LLM Prompt 工程，为用户提供从排盘到解盘的全链路一站式命理体验。

技术架构Technology

🔧 技术栈概览
  • 前端：HTML5 + Tailwind CSS + Three.js (3D 渲染) + Lunar-JS (农历计算)
  • AI 能力：主流 LLM API（GPT-4o / DeepSeek / Claude），多模型混合策略降本
  • 后端：Node.js / Python FastAPI，RESTful API 架构
  • 数据库：Supabase / MySQL，云端命例数据存储
  • 部署：Vercel / 阿里云，CDN 全球加速，前端离线缓存
🧠 核心算法能力
  • 紫微斗数引擎：含节气校正、农历转换、十四主星定位、四化飞星推演
  • 六爻算法：三铜钱起卦，真实物理概率模拟
  • 大运流年推算：从命盘结构推演一生起伏节点
  • 合盘评分模型：多维度量化评分，支持多关系类型
  • Prompt 工程：分场景 System Prompt 设计，输出格式约束
